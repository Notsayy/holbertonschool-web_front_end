## Présentation

Ce projet regroupe différents exercices et exemples pour découvrir et maîtriser le framework CSS [Bootstrap](https://getbootstrap.com/). Il permet de créer rapidement des interfaces web modernes, responsives et élégantes, sans écrire beaucoup de CSS personnalisé.

## Contenu du projet

- Utilisation du CDN Bootstrap
- Grilles responsives
- Composants (boutons, cartes, alertes...)
- Tableaux stylisés
- Exercices pratiques sur la mise en page et l'accessibilité

## Aperçu

Quelques exemples de ce que vous trouverez dans ce projet :

- Mise en page responsive avec le système de grille
- Utilisation des classes utilitaires Bootstrap
- Intégration de composants interactifs (modals, alertes, etc.)

## Démarrer

1. Ouvrez les fichiers HTML dans votre navigateur pour voir le rendu Bootstrap.
2. Modifiez, testez, amusez-vous !
