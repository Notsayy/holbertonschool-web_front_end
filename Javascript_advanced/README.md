# JavaScript Advanced - Holberton School

This project contains a series of advanced JavaScript exercises to practice and master modern JavaScript concepts. The exercises cover topics such as functions, classes, asynchronous programming, algorithmic challenges, and more.

## Project Content

- **0-welcome.js**: Introduction and basic setup.
- **1-nested_functions.js**: Working with nested functions.
- **2-function_me.js**: Higher-order functions and function manipulation.
- **3-classrooms.js**: Object-oriented programming and classes.
- **4-math.js**: Advanced mathematical operations.
- **5-mode.js**: Array and statistical operations.
- **6-hogwarts.js**: Context and binding in JavaScript.
- **7-timeout.js**: Asynchronous programming with timeouts.
- **8-payments.js**: Payment processing logic and callbacks.
- **9-prime.js**, **10-prime.js**, **11-prime.js**: Prime number algorithms and optimizations.
- **12-room_area.js**: Calculating areas and working with objects.
- **13-bind_user.js**: Function binding and user context.
- **14-wikipedia.js**: Fetching and processing data from external sources.

## Installation and Usage

1. **Clone the repository**

   ```bash
   git clone https://github.com/Notsayy/holbertonschool-web_front_end.git
   cd holbertonschool-web_front_end/Javascript_advanced
   ```

2. **Run the exercises**

   Each file can be run individually using Node.js:

   ```bash
   node <filename.js>
   ```

## Key Points

- All exercises use modern JavaScript (ES6+) features.
- Focus on best practices: code readability, modularity, and error handling.
- Each file is self-contained and can be run independently.

## Requirements

- Node.js (v12 or higher recommended)
